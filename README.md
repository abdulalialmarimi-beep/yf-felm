# بوت سينيما ديسكورد (Self-bot)

بوت يبث فيلم/مسلسل فعلي داخل Voice Channel، مع لوحة تحكم بأزرار وترجمة.

## ⚠️ تنبيهات مهمة
- يعمل بحساب مستخدم عادي (Self-bot) — يخالف شروط ديسكورد، استخدم حساب ثانوي قديم لا حسابك الرئيسي.
- مكتبة `discord.js-selfbot-v13` متوقف دعمها رسمياً (Archived) — تعمل حالياً لكن بدون تحديثات مستقبلية.
- الأزرار وميزة التقديم/الترجيع تجريبية، جرّبها وأرسل أي رسالة خطأ تطلع بالـ Logs لتصحيحها.

## المتطلبات
- Node.js 20+
- FFmpeg مثبت (أو استخدم Dockerfile المرفق عند النشر على Railway/Render)

## التثبيت محلي
```
npm install
```

## الإعداد
1. نسخة من `.env.example` باسم `.env`، وعبي:
   - `DISCORD_TOKEN`
   - `OWNER_ID` (آيدي حسابك، فقط هذا الحساب يقدر يستخدم الأوامر)
2. عدّل `library.json` بصيغة:
```json
{
  "اسم الفيلم": {
    "video": "رابط مباشر لملف الفيديو (mp4)",
    "subtitle": "رابط مباشر لملف ترجمة .srt (اختياري)"
  }
}
```
⚠️ لا تضع ملفات فيديو داخل الريبو على GitHub (الحجم الأقصى للملف حوالي 100MB) — استخدم روابط مباشرة فقط من مكان تخزين عندك.

## التشغيل
```
npm start
```

## الأوامر
| الأمر | الوظيفة |
|---|---|
| `!تشغيل <اسم>` | تشغيل فيلم/مسلسل من القائمة + لوحة تحكم |
| `!قائمة` | عرض الأسماء المتوفرة بـ library.json |
| `!ايقاف` | إيقاف التشغيل والخروج |
| `!مساعدة` | عرض الأوامر |

## لوحة التحكم (تظهر تلقائياً بعد !تشغيل)
- ⏪ 10 — رجوع 10 ثواني
- ⏸️/▶️ — إيقاف مؤقت / استمرار
- ⏩ 10 — تقديم 10 ثواني
- ⏹️ إيقاف نهائي

فقط حساب OWNER_ID يقدر يضغط الأزرار.

## النشر على Railway (عبر GitHub)
1. رفع الملفات (index.js, package.json, Dockerfile, library.json) على ريبو GitHub خاص — **لا ترفع `.env`**
2. اربط الريبو بـ Railway → New Project → Deploy from GitHub repo
3. أضف المتغيرات بـ Variables: `DISCORD_TOKEN`, `OWNER_ID`, `PREFIX`
4. بعد كل تعديل بـ library.json لازم ترفعه على GitHub، Railway يحدث تلقائياً.
